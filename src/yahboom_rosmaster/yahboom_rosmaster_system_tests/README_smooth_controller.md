# Smooth Keyboard Controller

## Controls

| Key | Action |
|-----|--------|
| **W** | Forward |
| **S** | Backward |
| **A** | Strafe Left |
| **D** | Strafe Right |
| **Q** | Rotate Left |
| **E** | Rotate Right |
| **X** | Stop |
| **SPACE** | Emergency Stop |

## Usage

```bash
ros2 run yahboom_rosmaster_system_tests smooth_keyboard_controller
```
